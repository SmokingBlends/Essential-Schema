Schema.org JSON-LD for Products, Returns, Shipping, and FAQs.

== Description ==
The **Essential Schema** plugin adds Schema.org structured data in JSON-LD format to your WooCommerce site. It enhances SEO by providing search engines with detailed information about products, returns policy, shipping details, and FAQs.

Key features:
- Product schema for WooCommerce products.
- Returns policy schema.
- Shipping details schema.
- FAQ schema.

The plugin loads module classes and includes a hidden settings screen accessible via a "Settings" link in the Plugins row. It ensures security with capability checks on the admin UI and sanitizes input while escaping output in module UIs.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/essential-schema` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings link on the Plugins page to configure if needed.

== Frequently Asked Questions ==

It adds essential Schema.org JSON-LD markup for products, returns, shipping, and FAQs to improve SEO.

A "Settings" link appears in the Plugins row after activation.
