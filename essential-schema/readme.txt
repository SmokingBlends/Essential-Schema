Schema.org JSON-LD for Organization, Products, Returns, Shipping, FAQs, Articles.

== Description ==
Adds Schema.org structured data in JSON-LD format to enhance SEO. Configurable via settings.

Key features:
- Organization schema on home.
- Product schema for WooCommerce.
- Returns and shipping details.
- FAQ schema on selected page.
- Article schema on posts.
- Custom item specifics.

== Installation ==
1. Upload to /wp-content/plugins/essential-schema/.
2. Activate.
3. Configure via Plugins row Settings link.

== Frequently Asked Questions ==
